# GymApp minimal backend

Servidor de prueba para desarrollo local. Implementa los endpoints listados en la especificación y usa almacenamiento en memoria.

Cómo usar:

1. Abrir terminal en la carpeta `backend`
2. Instalar dependencias: `npm install`
3. Iniciar servidor: `npm run start` o `npm run dev`

Por defecto el servidor corre en `http://localhost:3000/api/v1`.
